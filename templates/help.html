Ajuda

Como usar o Sistema de Login com Firewall

Como funciona
1. Registro

Para usar o sistema, você precisa criar uma conta fornecendo seu email e senha.

2. Confirmação

Após o registro, você receberá um email de confirmação. Clique no link para ativar sua conta.

3. Login

Faça login com suas credenciais. Seu IP será automaticamente liberado no firewall.

4. Logout

Ao sair, seu IP será removido do firewall automaticamente.

Suporte

Para dúvidas ou problemas:

  • Verifique os logs do sistema
  • Consulte a documentação
  • Entre em contato com o administrador
