Política de Privacidade

Coleta de Dados

Este sistema coleta apenas os dados necessários para seu funcionamento:

  • Email para autenticação
  • Endereço IP para controle de firewall
  • User Agent para identificação do dispositivo
Uso dos Dados

Os dados são utilizados exclusivamente para:

  • Controle de acesso via firewall
  • Logs de segurança
Segurança

Todos os dados são armazenados de forma segura e não são compartilhados com terceiros.
